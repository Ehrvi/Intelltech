# Data Directory

This directory stores validated, cleaned, and reusable datasets that are used across multiple analyses and reports.

## Files

- **`market_sizing_APAC.csv`**: Market size and growth data for key sectors across the APAC region.

- **`competitive_landscape.csv`**: A database of key competitors, their market share, strengths, and weaknesses.

- **`infrastructure_projects_APAC.csv`**: A list of major infrastructure projects in the APAC region, including budget, timeline, and status.

- **`README.md`**: This file.

**Usage:** Before conducting new research, check this directory to see if a relevant dataset already exists. Using these validated datasets saves time and ensures consistency across all analyses are based on the same source of truth.
