# Railways Sector Case Study: High Speed Rail Monitoring, South Korea

**Source:** G端ralp Systems - High Speed Rail Monitoring, Korea (Case Study - Seismology)

**Project Details:**
Project Name: Korail project network
Client: Korea Network Authority
Scale: Over 100 accelerometers and 65 digitisers at 34 seismic stations over 320 km of railway line
Technology: G端ralp strong-motion accelerometers with real-time data transmission
Purpose: Earthquake early warning system for high-speed railway infrastructure

**Summary:**
The Korail project network was formed by the Korea Network Authority to provide low-latency data for an early warning system. The solution consists of over 100 accelerometers and 65 digitisers distributed across 34 stations over 320 km of railway line. Each station contains two 5T tri-axial accelerometers connected to the network using DM24 digitizers and data modules.

**G端ralp Solution:**
The Korail project network consists of over 100 accelerometers and 65 digitisers spread at regular intervals over 320 km of railway line. G端ralp strong-motion accelerometers gather real-time data from structures along the newly-built high-speed railway line and transmit it to operators using a combination of Scream and GSMS protocols. GSMS can provide every second and on triggered data MMA (minimum, maximum and average ground acceleration), PGA (Peak Ground Acceleration), RMS (Root Mean Square), Cross correlation and SI (Spectral Intensity).

**Technical Implementation:**
A minimum of two accelerometers are used: one at the bottom of the column, one at the top. The instruments are connected to a single 6-channel DM24 mk3 digitizer in a nearby building, which is linked to the local TCP/IP network with a DCM. The DCM runs a Scream server so that real-time data can be viewed from anywhere on the network. Only one of the two digitizer input ports on each DCM is used, giving the array scope for future expansion as necessary.

**Key Features:**
Real-time monitoring of seismic activity and structural response
Low-latency data transmission for earthquake early warning
Comprehensive coverage over 320 km of high-speed railway infrastructure
Scalable architecture with expansion capability
Integration with national earthquake monitoring network
